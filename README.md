# KJRI Node.js app

1. Clone this repo
2. Install all dependencies

```shell
yarn
```

or with NPM

```shell
npm i
```

3. Start the app

```shell
yarn dev
```

or with NPM

```shell
npm run dev
```

## Deployment with Vercel

1. Make sure have a Vercel account
2. Deploy

```shell
yarn build && yarn deploy
```

or with NPM

```shell
npm run build && npm run deploy
```
